3. 图片

语法：

<img src="图片地址" alt="替换文本" title="图片标题" width="100" height="100" border="1">
注意图片地址可以是相对路径或者绝对路径，尽量使用相对路径
替换文本是指当图片无法显示时，浏览器会显示的文本内容
图片标题是指当用户将鼠标悬停在图片上时，浏览器会显示的文本内容

4. 超链接

语法：

<a href="链接地址" target="_blank">链接文本或者照片img</a>
target="_blank"表示在新窗口打开链接

效果：

(1).一个超链接（外部链接） 百度
(2).这也是一个超链接（内部链接） 我的第一个网页
(3).这也是一个超链接（空链接） 空链接（待做）
(4).这也是一个超链接（下载链接 下载zip或者运行exe等） 下载
(5).还可以做网页元素链接     链接到图片音频视频     或者用照片img等元素做链接等等
查看图片           

(6).还有锚点链接 返回顶部
锚点链接的地址是以#开头的链接，指向页面中的某个元素，需要在目标元素上设置相应的id属性。
