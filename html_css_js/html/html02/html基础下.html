1. 表格

表格是用来显示数据的一种常用方式

表格的基本结构和属性

语法：

    
    <table border="1" align="center" cellpadding="10" cellspacing="0" width="500" height="200">           
                                table表示表格，常用属性：
                                border：边框宽度
                                align：表格对齐方式(left/center/right)
                                cellpadding：单元格内边距
                                cellspacing：单元格间距
                    
        <thead>          thead表示表格头部区域
            <tr>               tr表示表格行
                <th>姓名</th>       th表示表头 是一种特殊的单元格
                <th>年龄</th>              要放在行tr里面
                <th>城市</th>                   内容会居中加粗
            </tr>
        </thead>
        <tbody>           tbody表示表格主体区域
            <tr>                      tr表示表格行 
                <td>张三</td>              td表示表格单元格
                <td>25</td>                    要写在行tr里面
                <td>北京</td>                     表示这一行有几个单元格
            </tr>
            <tr>
                <td>李四</td>
                <td>30</td>
                <td>上海</td>
            </tr>
            <tr>
                <td>王五</td>
                <td>28</td>
                <td>广州</td>
            </tr>
        </tbody>
    </table>

效果：


姓名 年龄 城市
张三 25 北京
李四 30 上海
王五 28 广州

合并表格单元格

语法：

rowspan：合并行
<td rowspan="2">内容</td>
colspan：合并列
<td colspan="2">内容</td>
注意合并之后删除多余的单元格即可

效果：

个人简历
姓名 张三 性别 男         照片       
年龄 25 城市 北京
民族 汉族 电话 12345678901
政治面貌 群众 学历 本科

2. 列表

列表是用来显示一组相关信息的常用方式 常用来布局

无序列表

语法：

<ul>
<li>项目1</li>
<li>项目2</li>
<li>项目3</li>
</ul>

效果：

  • 项目1
  • 项目2
  • 项目3

有序列表

语法：

<ol>
<li>项目1</li>
<li>项目2</li>
<li>项目3</li>
</ol>

效果：

  1. 项目1
  2. 项目2
  3. 项目3

自定义列表样式

语法：

<dl>
<dt>项目1</dt>
<dd>项目1的详细信息1</dd>
<dd>项目1的详细信息2</dd>
</dl>

效果：

项目1
项目1的详细信息1
项目1的详细信息2
