# MEMS-Enabled On-Chip Computational Mid-Infrared Spectrometer Using Silicon Photonics基于MEMS的硅光子芯片计算式中红外光谱仪

- 作者：Qifeng Qiao, Xinmiao Liu, Zhihao Ren, Bowei Dong, Ji Xia, Haoyang Sun, Chengkuo Lee*, Guangya Zhou*
- 发表：ACS Photonics, 2022, 9: 2367–2377（DOI: 10.1021/acsphotonics.2c00381）
- 关键词：MIR 光子学、硅光子、可重构光子学、光 MEMS、片上光谱仪
- 对象与场景：集成与便携场景下的生化光谱与高光谱成像；目标波段 3.7–4.05 μm
- 收获：想尝试在py中使用sklearn库使用网格搜索+交叉验证来进行正则化调参，但本文的调参思路比较普通，没有体现出py的优势，也没有很好的体现“网格搜索+交叉验证”。

## 概要
- 问题：MIR 波段片上光谱仪仍受限；传统基于色散元件与窄带滤波的多通道方案存在 SNR 与探测器阵列成本问题。
- 思路：在 SOI 平台上设计 MEMS 可调耦合器，通过“时间域调制耦合强度”以实现等效 OPD 的调制，获得干涉条纹（interferogram），并采用“机器学习-正则化回归”进行谱重构；单像素探测、具备 Fellgett（多路复用）与 Jacquinot（高通光）优势。
- 贡献：
  - 提出并实验演示片上计算光谱仪（Si-McS）于 4 μm 波段，MEMS 静电驱动，耦合强度可调，单像素读出。
  - 解析“耦合器两超模（SM0/SM1）干涉 → 通过/旁路端口输出干涉图 → 频谱重构”的等效 FT 机制，建立 x = L Δn(λ,h) 的 OPD 对应关系。
  - 采用**正则化回归**实现“稀疏激光谱”跨 350 nm 带宽的重构；在双波长情况下实现约 3 nm 分辨率；重构 N2O 气体吸收谱。
  - 设备指标：响应时间估计约 11.5 μs；波导损耗约 −2.9 dB/cm；弯曲损耗约 −0.076 dB；芯片占地 4.5 mm × 0.6 mm。

## 背景与挑战
- 传统片上方案：
  - 色散光栅/窄带滤波：高分辨率但分光到多通道，SNR 下降且需探测器阵列，成本与复杂度上升。
  - FT 方案：具有多路复用与高通光优势；但 MIR 区的器件与集成仍在发展中。
- MIR 片上可重构：MEMS 机械、自由载流子、热光等；其中 SOI 光 MEMS 可在去除 BOX 后实现波导机械运动并避免 BOX 在 MIR 的吸收损耗。

## 体系结构与工作原理
- 结构：悬置硅波导 + 亚波长光栅（SWG）包层；双波导耦合器（单/双侧 SWG 区分可动与固定波导）；MEMS 悬臂通过静电驱动改变“垂直间隙 h”。
- 超模干涉：
  - 耦合器形成两超模 SM0/SM1，输出端口（through/drop）的功率随“有效折射率差 Δn(λ)”与“耦合长度 L”变化而呈现干涉图：
    - P_through(λ,L) ≈ B(λ) · cos²(k L Δn(λ))
    - P_drop(λ,L) ≈ B(λ) · sin²(k L Δn(λ))
  - 对应于传统 FTIR 的 OPD x，等效关系：x = L · Δn(λ,h)。通过施加时间域偏置 V_bias 改变 h，从而步进调制 Δn。
- 获取干涉图：选用 through 或 drop 端口为读出，时间域扫描得到 interferogram，后端进行谱重构。

## 计算重构方法（弹性正则化回归）重点
- 目标：由干涉图 I（由不同 V_bias 下的采样点序列）重构输入谱 S（在目标离散波长栅格上的稀疏或稀疏近似向量）。
- 建模要点：
  - Δn(λ,h) 以多项式近似；将“电-机-光耦合调制 → 干涉图生成”的 forward 过程参数化，得到 I ≈ F(S)。
  - 采用带稀疏/稳健先验的“弹性正则化回归”求解欠定/病态的逆问题，兼顾稀疏峰保留与噪声鲁棒性。
- 训练与推断：
  - 以离散激光波长的测量干涉图作为训练集，网格搜索超参数（α1, α2）以平衡拟合与稀疏性。
  - 稀疏激光谱重构：跨 350 nm 带宽（3.7–4.05 μm）重构达到单/双峰情形，双峰示例分辨率约 3 nm。
  - 多色假设：基于“线性叠加”的探测器响应假设，将多波长干涉图表示为各单色干涉图叠加，进行多色谱估计。
- 噪声与稳健性：对实验干涉图施加不同噪声水平评估稳健性，正则化回归在合理 α 下保持可识别谱峰与吸收特征。
- 思考：由于机器学习和光谱重建底层数学原理有所不同，导致在本文中sklearn库没有很好的发挥作用。但是网格搜索和交叉验证的思路仍然可以使用。

## 结果与性能
- 带宽与分辨率：
  - 激光稀疏谱重构带宽约 350 nm；双峰重构分辨率约 3 nm（示例 Δλ = 2–3 nm）。
- 吸收谱：
  - N2O 气体吸收谱重构与识别（在气室测试下），表明在 MIR 的生化感知应用潜力。
- 硬件指标：
  - 响应时间约 11.5 μs；波导与弯曲损耗小；单像素探测；芯片小型化（4.5 × 0.6 mm）。
- SNR 优势：
  - 继承 FT 的多路复用与高通光优势，相比多通道分光，有利于提升 SNR。
